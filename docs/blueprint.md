# **App Name**: BLMC Connect

## Core Features:

- Secure User Authentication & Roles: Robust Firebase Email/Password authentication supporting Admin, Staff, and Member roles, including a registration approval process, password reset, and secure logout.
- Cooperative User Management: Admin users can approve/reject new registrations, create, update, and deactivate user accounts. Members have the ability to update their personal information, with all user data stored in Firestore.
- Dynamic Announcements, Events & Notifications: Administrators can create, read, update, and delete announcements and events, which members can view in real-time. Basic notifications will alert users to new updates.
- Personalized Dashboard: A tailored dashboard view for each user role, displaying relevant announcements, individual savings balance, debt balance, and notifications, along with quick access buttons to common actions.
- Online Membership Registration: An online form for prospective members to submit applications, featuring an admin response system for managing applications and tracking registration status.
- AI Member Assistant Tool: A chat interface powered by generative AI that acts as a tool to answer frequently asked questions, guide members through cooperative processes like borrowing, and facilitate basic inquiries into their savings and ledger status.
- Basic Savings & Debt Tracking: Displays members' current savings balance and debt status. Includes functionality to record new deposits and payments to update balances.

## Style Guidelines:

- A light color scheme to convey clarity and professionalism, with a muted purple-grey as the primary hue (e.g., #69568F) symbolizing trust and stability for the cooperative.
- Background color uses a very light, desaturated variant of the primary hue (e.g., #F1F0F4) for a clean and unobtrusive backdrop.
- An analogous, slightly brighter accent color (e.g., #7082DB) is used for call-to-action buttons and interactive elements, providing clear contrast and emphasis.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for a professional management system.
- Use a set of clear, line-based or filled professional icons that intuitively represent cooperative functions, financial actions, and user management across the application.
- Implement a structured, clean, and intuitive layout designed for ease of use, prioritizing readability and clear information hierarchy across all role-based dashboards and forms. Ensure full mobile responsiveness.
- Incorporate subtle and swift animations for transitions, form submissions, and user feedback, providing a smooth and responsive user experience without being distracting.
